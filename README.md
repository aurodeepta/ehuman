# eHuman

Advanced Android Assistant with Camera, Face Detection, Voice Interaction, and LLM Chat.

## Features
- Live Camera with Real-time Face Detection (ML Kit)
- Long-press Volume Up → Read trending news with TTS
- Real-time Face Emotion Analysis during TTS
- Continuous Speech-to-Text → Feeds into Chat Window
- LLM-ready Chat (placeholders for Grok, Gemini, OpenAI)

## Tech Stack
- Kotlin + Jetpack Compose
- CameraX + ML Kit
- TextToSpeech + SpeechRecognizer

Clone and open in Android Studio.
